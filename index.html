AES

The Advanced Encryption Standard (AES) Implementation project involves developing a cryptographic system in Python to encrypt and decrypt data using the AES algorithm. AES is a symmetric block cipher that processes 128-bit data blocks with encryption keys of 128, 192, or 256 bits. The implementation includes key components such as Key Expansion, which generates round keys, and fundamental transformations like SubBytes, ShiftRows, MixColumns, and AddRoundKey, ensuring secure and efficient encryption. The system supports both encryption and decryption and has been tested using official AES test vectors to verify its correctness.

FPGA Checksum Verifier

This project involves the creation of an FPGA-based digital module that performs IP header checksum verification, as specified by RFC 791. The module, developed in VHDL, receives IP packet headers, performs checksum calculations, and verifies the results against the received checksum. The system tracks valid and invalid checksums and reports them using LED indicators. A testbench simulates the processing of IP packets, reading data from an input file and systematically testing the module. This project offers practical experience in FPGA development, VHDL programming, and the design of digital systems for communication networks.

Smart Home Enviromental Control

This project automates indoor climate regulation using an ESP8266 microcontroller, RFID authentication, and IoT connectivity. Users scan an RFID chip to retrieve their preferred temperature and humidity settings, which are stored on a backend server. Sensors measure real-time conditions, and adjustments are made via a servo-controlled radiator valve, a fan, and an LED indicator. The system communicates via Wi-Fi using MQTT, allowing users to monitor and modify settings through a web application. Tests confirmed accurate sensor readings and effective climate control. Future improvements could include offline functionality by storing presets directly on RFID chips.

Secure TicTacToe

This project involves the development of a network-based Tic-Tac-Toe game with secure communication. Bellman-Ford

This project involves the implementation of the Bellman-Ford algorithm for shortest path computation in a weighted directed graph. Developed in Python, the system initializes node distances and iteratively updates them by processing all edges, ensuring accurate path calculations while detecting negative-weight cycles. The algorithm's execution is visually represented using NetworkX and Matplotlib, displaying nodes, edges, and updated distances in real-time. The system provides a structured visualization of shortest paths and network topology, offering a clear representation of the algorithm's operation.
